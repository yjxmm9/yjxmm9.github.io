**作品：**《基于AoIP技术的超高清电视转播系统应用》

**作者：**杨柳青 

**专业：**数字媒体技术

**指导老师：**严伟

**创作时间：**2024

**作品简介：** 

在广播电视音频领域，随着各类转播项目对于音频信号的体量需求的攀升，传统基带转播车的架构已经无法很好的完成大体量的转播项目。而借助着互联网技术的发展，转播系统不断地向IP系统迭代。本文结合笔者参与海南蓝湾LPGA高尔夫大师赛的转播实践经历，深入探索IP系统在实际转播过程当中的应用，以及AoIP技术的优势和发展方向。
